# Todo CLI

Pequeña aplicación de línea de comandos para gestionar una lista de tareas (todo).

Características:
- Añadir tareas
- Listar tareas
- Marcar tareas como completadas
- Eliminar tareas

Persistencia: las tareas se guardan en `todos.json` en la misma carpeta.

Uso:
- Añadir: `python todo.py add "Comprar leche"`
- Listar: `python todo.py list`
- Marcar completada: `python todo.py done 2` (usa el índice mostrado en `list`)
- Eliminar: `python todo.py remove 2`

Este proyecto sirve para practicar: archivos, listas, diccionarios, funciones y argumentos CLI.
